# Sarte Global - Goedkope Website Laten Maken

**Next.js 16** — Modern, high-performance website for Sarte Global, a digital agency specializing in affordable web development.

## 🚀 Tech Stack

| Category       | Technology                              |
| -------------- | --------------------------------------- |
| **Framework**  | Next.js 16 (React 19)                  |
| **Styling**    | Tailwind CSS 4 + PostCSS               |
| **Animation**  | Framer Motion 11                       |
| **Icons**      | React Icons                            |
| **Language**   | TypeScript 5                           |
| **Linting**    | ESLint 9                               |

## ✨ Features

- Responsive, mobile-first design
- Server-side rendering with Next.js App Router
- Smooth scroll-triggered animations (Reveal, Counter, Carousel)
- Dynamic team showcase with interactive 3D-like scenes
- Service cards, process steps, testimonials, and blog/journal section
- Cookie consent bar
- Contact form dialog
- Newsletter signup
- Optimized images with AVIF/WebP support
- Strict TypeScript for type safety
- ESLint best practices

## 🛠️ Getting Started

```bash
# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build

# Start production server
npm start

# Lint
npm run lint

# Type check
npm run typecheck
```

## 📁 Project Structure

```
app/             — App Router pages and layouts
components/      — Reusable UI components
constants/       — Data and content constants
contexts/        — React contexts (ContactDialog, ArticleDialog)
data/            — Static data files
hooks/           — Custom React hooks
lib/             — Utility libraries
public/          — Static assets (images, favicons, etc.)
types/           — TypeScript type definitions
scripts/         — Build scripts (favicon generation, icon analysis)
```
